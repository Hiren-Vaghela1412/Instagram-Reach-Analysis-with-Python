# 📸 Instagram Reach Analysis using Python

A complete **Data Analytics and Machine Learning project** that analyzes Instagram post performance, engagement metrics, and reach insights using **Python, Data Visualization, and Predictive Modeling**.

---

## 🚀 Project Overview

This project analyzes Instagram engagement data to uncover meaningful insights about post reach and audience interaction.

The analysis focuses on:

* Understanding Instagram reach sources
* Measuring engagement performance
* Identifying relationships between engagement metrics
* Calculating follower conversion rate
* Predicting post impressions using Machine Learning

This project demonstrates practical use of **Exploratory Data Analysis (EDA), Data Visualization, and Regression Modeling** for social media analytics.

---

## 📂 Dataset

**Dataset Used:** `Instagram data.csv`

The dataset contains Instagram post analytics such as:

* Impressions
* Likes
* Comments
* Shares
* Saves
* Profile Visits
* Follows
* Reach from Home
* Reach from Hashtags
* Reach from Explore
* Captions
